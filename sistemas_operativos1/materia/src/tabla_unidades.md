# Tabla ISO

| Valor | Bytes                  |
| ----- | ---------------------- |
| 1 KB  | 2^10 bytes= 1024 bytes |
| 1 MB  | 2^20 = 1024 KB         |
| 1 GB  | 2^30 = 1024 MB         |
| 1 TB  | 2^40 = 1024 GB         |
| 1 PB  | 2^50 = 1024 TB         |

# Ejemplo

1 GB = 2^30 = 2^30
2 KB = 2\*2^10 = 2^11
Total = 2^30 / 2^11 = 2^19
Bytes = 2^19/2^3 = 2^16
KiloBytes = 2^6 \* 2^10 = 64\*1 = 64 KB

> 1 GB con bloques de 2 KB es igual a 2^19 bloques (bits)
